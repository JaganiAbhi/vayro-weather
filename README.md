# VAYRO

A modern, clean, and production-ready Weather App built with React and Tailwind CSS.

## Features

- Real-time weather information
- Hourly and 7-day forecasts
- Weather details (humidity, wind speed, pressure, visibility, sunrise/sunset)
- Dark mode support
- Mobile-first responsive design
- Search by city name or use geolocation

## Tech Stack

- React
- Tailwind CSS
- Vite
- OpenWeatherMap API

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the root directory and add your OpenWeatherMap API key:
```
VITE_WEATHER_API_KEY=your_api_key_here
```

3. Start the development server:
```bash
npm run dev
```

## Build

To build for production:
```bash
npm run build
```
